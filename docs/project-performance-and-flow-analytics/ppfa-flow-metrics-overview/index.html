Flow Metrics Overview

The Flow Metrics Overview section provides a complete analysis of how issues move through the workflow during the selected period. Its purpose is to help identify bottlenecks, unused or unnecessary states, state frequency patterns, and overall flow efficiency. With summary cards, comparative charts, and a detailed list view, this section helps validate the true behavior of the workflow and highlights opportunities for optimization.
